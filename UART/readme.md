# PRÁCTICA UART CON DOS PLACAS FPGA

Por medio de protocolo UART se mandan datos eentre una tarjeta FPGA transmisora y una tarjeta FFPGA receptora.
Se defie un dato numérico por medio de los switches, el usuario debe presionar un botón para mandar ese dato y se debe visualizar en los displays de la placa receptora, y sin necesidad de resetearlo, se debe poder mandar y recibir otro dato cualquiera.
Es importante cablear las tierras de ambas placas para poder cerrar el circuito de la comunicación.
