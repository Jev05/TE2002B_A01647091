# PRÁCTICA CONTRASEÑA


Usando la lógica de una máquina de estados, se toma una contraseña ingresada por medio de los switches para evaluar si cada uno de los dígitos es correcto, en el orden correspondiente.

El usuario ingresa un número por medio de los switches, y para poder cargarlo debe presionar un botón, y así sucesivamente Si en alguno de los dígitos se equivoca, sin importar si no ha terminado de ingresar la contraseña, imprimirá la palabra bad y tendrá que resetearlo, si la contraseña es correcta, imprimirá good.
