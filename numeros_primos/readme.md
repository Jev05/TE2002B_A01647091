# PRÁCTICA NÚMEROS PRIMOS

Se tomaron 4 switches para representar los números decimales del 0 al 15, y por medio de un LED indicar si el número representado es primo no.

Si se representa el número 1101 en binario, 13 en decimal, el LED enciende, pues el número 13 es un número primo.

En la lógica, los números 2, 3, 5, 7, 11 y 13 son determinados como primos, y debe encender el led indicando esa propiedad.

Se deja que el default es el led apagado, para que el led solo encienda si es primo.
