# PRÁCTICA BCD CON DISPLAY DE 7 SEGMENTOS

En esta práctica se programan dos módulos para que, por medio de switches, se puedan decodificar los números de binario  decimal y mostrarlos en los displays de 7 segmentos.

El módulo segments se definen los segmentos de un diplay que deben encender depediendo la cantidad que se ingrese en los switches. Este módulo se instancia para cada uno de los displays.
En el módulo BCD_4displays, se hace la separación de los dígitos que representarán cada display, es decir, toma el valor que se ingrese con los switches, y los divide en unidades, decenas, centenas y unidades de millar para poder representarlos individualmente en cada display.
