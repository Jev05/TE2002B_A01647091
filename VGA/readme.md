# PRÁCTICA VGA

Se segmenta la pantalla para poder mostrar rectángulos de colores definidos por el código RGB.
Además, se realiza un tablero de ajedrez mapeando y haciendo de la pantalla una cuadricula que se define de multiplicar el contador en X por el contador en Y, que vienen siendo los espacios disponibles antes de que se "acabe" la pantalla.
